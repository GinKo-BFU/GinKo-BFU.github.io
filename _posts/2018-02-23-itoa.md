## itoa

### Experience
做进制转换题偶然发现了itoa函数，它能把转换后的结果存到字符数组中。
所以觉得这个不错，它能把数字打成字符，就不用自己除了

像这样
``` cpp
//输出32

#include<iostream>

#include<cstdlib> 

using namespace std;
int main()
{
	char a[5];
	itoa(32,a,10);
	cout<<a[0]<<a[1];
	return 0;
}
```
